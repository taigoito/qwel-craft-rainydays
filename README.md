# Rainy Days | Created by ― QWEL.DESIGN

東京都町田市在住の頃、雨水を貯めて活用するという思想の元、ごく小さな規模で市民活動を行っていた。  
そのとき、教育活動の一環として使えないかと思い付きでつくった「あめぐらゲーム」のプロトタイプ。
